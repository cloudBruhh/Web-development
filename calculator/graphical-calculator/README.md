# Graphical Calculator

This project is a graphical calculator that provides a user-friendly interface for performing basic arithmetic operations. It includes buttons for digits and operations, as well as a display area for the calculator.

## Project Structure

```
graphical-calculator
├── src
│   ├── index.html       # Main HTML document for the calculator interface
│   ├── styles.css       # Styles for the calculator layout and design
│   └── script.js        # JavaScript code for calculator functionality
├── assets
│   └── fonts            # Directory for font files used in the design
└── README.md            # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in a web browser to view the calculator.

## Usage Guidelines

- Click on the buttons to input numbers and operations.
- The display area will show the current input and results of calculations.
- Supports basic operations: addition, subtraction, multiplication, and division.
